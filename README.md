# A Little Introduction

This repository serves as a navigation tool for my portfolio of java projects. The following paragraphs will introduce the projects and provide links to them.<br><br>

## 1 Tank War Game ([demo video](https://youtu.be/3K--jC5hLa4))

A tank game where the player controls a tank to fight multiple enemy tanks. Supports save and load.<br><br>
Modules: Player Control; Enemy AI (randomly moves and shoots); Collision Detection; Rendering (with JPanel); Game State Management; Animation; Event Handling; Thread Management.<br><br>
You can find it [here](https://github.com/GentleAmber/Tank-game).

## 2 Instant Messaging System ([demo video](https://youtu.be/nJLmUAOUesg))

A messaging system allowing multiple clients to exchange and broadcast messages, as well as send files.<br><br>
Modules: Command-line Menu; Client; Server; File Transfer; User Management; Network Communication.<br><br>
You can find it [here](https://github.com/GentleAmber/Instant-messaging-system).
